# 🏨 Bondo Inn

Bondo Inn is a premium hospitality booking platform designed to help guests discover, explore, and book exceptional accommodations through a beautiful, responsive, and modern interface.

## ✨ Features

- 🏨 Browse featured rooms and suites
- 🔍 Search and filter accommodations
- 📍 View hotel location and nearby attractions
- 🛏️ Detailed room information
- 📱 Fully responsive design
- 🎨 Elegant modern UI
- ⚡ Fast and optimized performance

## 🚀 Tech Stack

- HTML5
- CSS3 / Tailwind CSS
- JavaScript
- React
- TypeScript
- Vite

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/bondo-inn.git
```

Navigate into the project:

```bash
cd bondo-inn
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
